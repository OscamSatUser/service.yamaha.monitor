# service.yamaha.monitor

Switches Yamaha Receiver DSP mode.
2 audio channels or less sets 7.Channel.Stereo otherwise sets straight mode.

Test on Yamaha RX-681 Multicast Receiver and older RX-671 YNC (Yamaha Network Control) receiver.
No AVinfo screen support for YNC models.

Allows for optional video pause seconds at start to allow TV time to refresh screen. Only applicable to Local or SMB video files.
Optional AVinfo screen display (multicast receivers)
