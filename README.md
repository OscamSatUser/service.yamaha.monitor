# service.yamaha.monitor
Switches Yamaha Receiver DSP mode between straight and 7.Channel.Stereo based on # of audio channels. Yamaha receiver must support yamaha extended control (musiccast) api.

Works with video or audio channels.  Settings allows you to specify IP address of yamaha receiver, whether or not to show Yamaha Video Info screen.  How many seconds to wait to show info screen, and for how long to show info screen.

Between TV, AV receiver negotiating the video resolution and refresh rate, it can take a bit of time to sort out and actually start showing video on the screen.  The pause allows you to get past all that so that you can see the Video Info Description.
